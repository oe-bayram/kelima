Read the `DESIGN.md` file within this folder, and explore the other available files (`tokens/`, `components/`, `ui_kits/app/`, `assets/`).

Key facts:
- German UI, informal „du"; no emoji; Lucide icons.
- Cool ice neutrals, Kelima ocean-blue primary (navy ink #232B57), teal-green accent, didactic 5-step learning-status colors (Neu/Nicht gewusst/Schwer/Kann ich/Sicher → gray/red/amber/teal-green/navy-blue). Brand gradient only for brand moments.
- Display = Schibsted Grotesk, Text = Hanken Grotesk, Mono = JetBrains Mono.
- The Lemma (vocabulary word) is always the visual hero.
