Read the `DESIGN.md` file within this folder, and explore the other available files (`tokens/`, `components/`, `ui_kits/app/`, `assets/`).

Key facts:
- German UI, informal „du"; no emoji; Lucide icons.
- Warm paper neutrals, pine-green primary, didactic 5-step learning-status colors (Neu/Nicht gewusst/Schwer/Kann ich/Sicher → gray/red/amber/green/blue).
- Display = Schibsted Grotesk, Text = Hanken Grotesk, Mono = JetBrains Mono.
- The Lemma (vocabulary word) is always the visual hero.

> This is the **local mirror** of the claude.ai/design project
> "Lernwort — B1 Vokabeltrainer Design System" (id bca96c9e-…). It is the
> source of truth for the app's look; you no longer need to open the design
> project to recall values or component behaviour. The compiled web bundle and
> HTML preview harnesses were intentionally not mirrored (see INDEX.md).
